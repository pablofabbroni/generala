# Generala Counter MVP

MVP: Setup de partida (1-4 jugadores + colores + variantes) + contador manual de puntajes + ranking final.

## Requisitos
- Node.js 18+ (recomendado 20+)

## Instalación
```bash
npm install
npm run dev
```

Abrí http://localhost:3000

## Notas
- Estado persistido en `localStorage` (podés reiniciar desde Home).
- Puntaje es **manual**: tocás una celda vacía y cargás el número.
